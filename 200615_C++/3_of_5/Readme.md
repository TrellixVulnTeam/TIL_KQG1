## 클래스 상속

> 기존에 정의된 클래스 멤버(변수, 함수)를 상속받아 새로운 멤버를 추가할 수 있는 기능

- 효율적 **유지보수** 가능

- 구성

  - 기본클래스 (상속해주는)
  - 파생클래스 (상속받는)

- 파생 클래스 선언 형식

  ```c++
  class 파생클래스명 [접근변경자] 기본클래스명 {
    [추가할 멤버 선언;]
    // 1. 상속받는 멤버
    // 2. 재정의한 멤버함수
    // 3. 새로 추가한 멤버 
  }
  ```
